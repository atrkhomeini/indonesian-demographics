# Indonesia Demographics Market Intelligence

Interactive dashboard for analyzing Indonesian demographic trends and market segmentation.

## Features

- **Market Segmentation**: Quadrant analysis based on TFR and expenditure
- **Time Series Forecasting**: ARIMA-based expenditure projections
- **Regional Analysis**: Deep-dive into individual markets
- **Data Explorer**: Browse and download datasets

## Data Sources

- Total Fertility Rate: SP2020 Long Form (BPS)
- Expenditure per Capita: Susenas 2010-2025 (BPS)

## Tech Stack

- Python, Pandas, Streamlit
- Plotly for interactive visualizations
- ARIMA for time series forecasting

## Usage

Navigate through different analysis pages using the sidebar menu.
